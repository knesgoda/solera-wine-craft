# Solera — Winery Management Platform

## Sitemap Proxy

The dynamic sitemap is generated by the `sitemap` Edge Function, which queries published blog posts from the database.

**Production hosting must proxy `/sitemap.xml` to the Edge Function:**

```
/sitemap.xml → https://vibdxsntxwrepnkrlbdw.supabase.co/functions/v1/sitemap
```

Configure this as a redirect/rewrite rule in your hosting provider (Netlify `_redirects`, Vercel `vercel.json`, Cloudflare Pages `_redirects`, etc.). The `robots.txt` already points to `https://solera.vin/sitemap.xml`.
